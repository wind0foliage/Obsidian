# 术语

**SN码**，全称为 **Serial Number**，中文通常译为 **“序列号”** 或 **“产品序列号”**。

# 场景：

针对贵重物品比如手机等3C产品客户，货品上拥有全球唯一码（SN码）的客户，用 SN 码帮助客户监控货品的流转，可以对货品进行一对一的追踪。

您可以把它理解为 **一个产品的“身份证号码”**。它的核心特性是 **唯一性**：全球范围内，任何一个合法的、带有SN码的产品，其SN码都是独一无二的，绝不会重复。


# **wms系统中包含两种SN码 种类：**

强SN码 ： 出库、入库都需要校验
弱SN码：  只有出库的时候需要校验

针对数码产品，手机产品可以使用**强序列号**进行管理，**因为具有唯一性，可追踪性，方便查询以及管理**。

针对高档服装等货品可以推荐使用**弱序列号**，商家准备好一批货品，然后出库时记录序列号，**能追踪具体订单**

# 功能说明


**强序列号：** 入库（开单必须扫描序列号）>> SN码管理（状态：已入库）>> 出库（验货必须扫描序列号）>> SN码管理（状态：已出库）

**弱序列号：** 出库（验货必须扫描序列号）>> SN码管理（状态：已出库）



# 实现原理

**【SN码的全球唯一性】 = 【身份前缀】× 【唯一后缀】**
