查看每一个线程的执行时间：

SELECT ID, USER, HOST, DB, COMMAND, TIME, STATE, INFO FROM information_schema.processlist where DB = 'd_tcsp_wms';

|        |                         |
| ------ | ----------------------- |
| `TIME` | 表示该线程的状态（STATE）已经持续了多少秒 |


推荐排序方式：按 `TIME DESC` 降序排列
SELECT NOW() AS check_time, ID, USER, HOST, DB , COMMAND, TIME, STATE , LEFT(INFO, 1000) AS sql_text FROM information_schema.processlist WHERE COMMAND != 'Sleep' AND INFO NOT LIKE '%information_schema%' AND (INFO IS NULL OR INFO NOT LIKE '%NOW%') ORDER BY TIME DESC;
